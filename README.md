# Tectum Soluções Ergonômicas — Landing Page

Uma landing page B2B premium, focada em conversão, desenvolvida para uma consultoria especializada em ergonomia, saúde ocupacional e adequação à NR-1.

## 🚀 Tecnologias Integradas
- **HTML5 & CSS3** (Vanilla, sem frameworks pesados)
- **Design System Dinâmico**: Tipografia moderna (Sora e DM Sans) e paleta corporativa de alto impacto (Laranja, Prata e Dark Navy/Charcoal).
- **Zero Config**: Fontes e ícones carregados via CDN (Google Fonts / SVG natevo).

## ✨ Principais Funcionalidades Visuais
- **Typewriter Animation**: Efeito de máquina de escrever real no título principal, com cursor piscante nativo.
- **Glassmorphism & Bento Grid**: Cards translúcidos e design em grid assimétrico (inspirado no shadcn).
- **Cartão do Especialista 3D**: Foto frameless da fundadora com rastreamento do cursor do mouse (efeito de parallax 3D) e drop-shadow realista.
- **Microinterações Avançadas**: Efeito de botão magnético ("magnetic button"), pulso de CTA e "spotlight" acompanhando o mouse nos cards de soluções.

## 🔧 Como Testar (Rodando Local)
Nenhum build step é necessário. Basta clonar o repositório e abrir o arquivo `index.html` em qualquer navegador moderno.

```bash
git clone https://github.com/yurizinlala/jusolucoes.git
cd jusolucoes
# Abra o index.html no Chrome / Edge / Firefox
```

## 📐 Estrutura de Diretórios
- `index.html`: A estrutura principal da página e todo o copywriting focado em conversão.
- `style.css`: Estilizações globais, sistema de cores, animações em keyframes e personalização de scrollbar.
- `script.js`: Toda a interatividade da página — scroll reveals (IntersectionObserver), accordion do FAQ, formulário de captura e matemática das animações do mouse.
